# Developmental-Mapping-of-GPR92-in-Human-Neonatal-Pancreas-and-chronic-pancreatis-
Performed scRNA-seq and spatial transcriptomics integration of human neonatal pancreas to define developmental-stage–specific GPR92 expression patterns. Applied trajectory inference and spatial alignment using Tangram to map GPCR-associated immune and stromal niches within anatomical context.
